The Basics of the Web

World Wide Web

  • collection of HTML (hyper text markup language) documents
  • hyper links connect documents
  • HTML is how you link content (images, etc)
  • Pieces of content
    • your browser makes requests via internet to servers using HTTP, response is file that browser displays
    • server - computer optimized to host/hold files

Hyper Text Markup Language

  • text content - you see
  • mark up - arrangement
  • references to other docs - images, videoes (website has a reference to youtube video in middle)
  • Mark Up
    • tag = <tag name>
    • <name> content </name> ---> called an element
    • bold <BOLD>
    • italic <EMPHASIS>
    • forgot </endtag>? Tag effects everything after opening tag
  • HTML Attributes
    • atrributes = ATTR
    • <tag ATTR="value"> contents </tag>
    • anchor tag <a> = to make a link
    • <a href="www.reddit.com"> derp </a>
      • derp would be the link!
    • image tag <img src="url" alt="text">
      • alt="text" is what is displayed if the image doesn't load - common courtesy
    • VOID TAG - no content = no closing tag
  • Whitespace
    • <br> VOID TAG - creates new line (inline - just creates a new line to work on/use as separation)
    • <p> </p> - creates some separation (makes an invisible box around that text)
    • why have both? <p> </p> can be used to create height
    • inline vs block
  • Span/div - to contain content
    • span - inline <span class="foo"> text </span>
    • div - block <div class="bar"> text </div>
      • will create a box around text, a separation like <p> </p> does
  • HTML Document
    • <DOCTYPE HTML> doctype
    • <html>
    • <head> place to put metadata, css, java
      • <title> Title of the webpage</title>
    • </head>
    • <body>
    • <b>content</b>
    • </body>
    • </html>
